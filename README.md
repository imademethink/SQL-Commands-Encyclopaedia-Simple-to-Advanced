To practice DB commands:

1) Install docker on your system
2) Run the docker-compose command below (docker-compose.yml file attached in this repo)

    cd SQL-Commands-Encyclopaedia-Simple-to-Advanced

    docker-compose up -d

3) Connect to DB using command line or any tool (I used DBeaver)
4) Connection url - localhost:5432, user/pwd - admin/ password123
